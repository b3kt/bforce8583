Output Types and Platform-Compiler combinations

Output Types

kdevelop - KDevelop *.kdevprj files.

make - GNU make files.

msvstudio - Microsoft Visual Studio workspace files (*.dsw) if subdir template, project files (*.dsp) otherwise.

nmake - Microsoft Visual Studio nmake files.
